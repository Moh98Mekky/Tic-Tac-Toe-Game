# 🎮 Tic Tac Toe Game (C# WinForms)

[![Status](https://img.shields.io/badge/Status-Completed-brightgreen)]()
[![Framework](https://img.shields.io/badge/Framework-.NET-blue)](https://dotnet.microsoft.com/)
![C#](https://img.shields.io/badge/Language-C%23-purple)
![Windows Forms](https://img.shields.io/badge/UI-WinForms-blueviolet)

A modern desktop implementation of the classic Tic Tac Toe game built using C# and Windows Forms.

---

## 📝 About the Project

This project was developed to practice:

- C# Programming
- Windows Forms Development
- Game Logic
- Event Handling
- UI Design

The game allows two players to play on the same computer using a clean and interactive interface.

---

## ✨ Features

- 🎮 Two-player gameplay
- 🧠 Winner detection
- 🤝 Draw detection
- 💡 Clean UI Design
- ⚡ Real-time interaction
- 🖱️ Easy navigation
